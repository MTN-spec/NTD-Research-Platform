📖 Methodology Summary

Phase I — The Eye

Sentinel-2 MSI Level-2A imagery. Bands B4, B8, B11, B12. Indices: NDVI, NDWI, SAVI. Cloud masking via SCL.

Phase II — The Brain

12 features × 480 observations. 70/30 stratified split. Z-score scaling. Random Forest & SVR with GridSearchCV.

Phase III — The Voice

4-tier threshold classification. Automated email alerts via SMTP. Interactive Leaflet web dashboard.
