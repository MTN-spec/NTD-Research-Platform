# Neural Tube Defect Prediction Model\n\n## Overview\nThis project implements a Machine Learning pipeline to predict the risk of Neural Tube Defects (NTDs) by integrating genomic (1000 Genomes) and transcriptomic (GTEx) data. \n\n## Project Structure\n```\n.\n├── data/\n│   ├── raw/             # Raw VCF and GCT files\n│   └── processed/       # Cleaned genotype matrices\n├── notebooks/\n│   ├── 01_Data_Acquisition.ipynb    # Download scripts for MTHFR, DHFR, FOLR1\n│   ├── 02_Data_Preprocessing.ipynb  # VCF parsing and expression filtering\n│   ├── 03_Model_Development.ipynb   # ML Models (Random Forest, XGBoost, DNN) + Synthetic Data Generator\n│   └── 04_Validation.ipynb          # SHAP analysis and Biomarker identification\n├── requirements.txt     # Python dependencies\n└── README.md\n```\n\n## How to Run\n\n### Option 1: Google Colab (Recommended)\nSince the datasets are large and require significant bandwidth, running on Google Colab is recommended.\n1.  Upload the `notebooks/` folder to your Google Drive.\n2.  Open `01_Data_Acquisition.ipynb` in Colab to download the real data.\n3.  Open `03_Model_Development.ipynb` to train the models. \n    - **Note:** You can run `03_Model_Development.ipynb` **immediately** using the built-in *Synthetic Data Mode* to verify the code and logic without waiting for downloads.\n\n### Option 2: Local Execution\n1.  Install Python 3.8+.\n2.  Install dependencies:\n    ```bash\n    pip install -r requirements.txt\n    ```\n3.  Launch Jupyter:\n    ```bash\n    jupyter notebook\n    ```\n\n## Objectives Covered\n1.  **Data Acquisition**: Automated scripts for 1000 Genomes and GTEx.\n2.  **Analysis**: Preprocessing pipelines for multi-omics data.\n3.  **Model**: Implementation of Random Forest, XGBoost, and Deep Neural Networks.\n4.  **Validation**: Model evaluation using AUC-ROC and feature importance analysis (SHAP).\n
